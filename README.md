
# Saloon Yehansa Backend

Saloon Yehansa Backend is the server-side application developed for managing saloon operations, customer bookings, authentication, and administrative functionalities.

## Features

- REST API development
- User authentication using JWT
- Role-based access control
- Secure password encryption
- Forgot password with OTP email verification
- Appointment booking management
- Duplicate booking prevention system
- Bridal package booking restrictions
- Service management APIs
- Customer review and rating management
- AI chatbot backend integration
- Admin dashboard management
- Email notification system
- MongoDB database integration
- Error handling and validation

## Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Nodemailer
- bcrypt.js
- dotenv
- CORS
